SmartCalc v1.0

Создание программы калькулятора расширенного вида.

С помощью разработанной программы можно построить график с значением Х, решить уравнения,
выражения, рассчитать кредит.

Структура проекта:

В директории lib находится реализация калькулятора через польсую нотацию в расширении *.c 

В директории Calculator находятся файлы создания GUI и построение графиков в расширении *.cpp

make install - создает программу калькулятора

make uninstall - удаляется все созданные файлы
